# ExamManagementSystem
github 远程配置

1. 创建`github` 仓库

2. 然后使用

```txt
git clone https://github.com/180CM55KG/ExamManagementSystem
```

将仓库克隆到本地

3. 将项目复制过来

4. 使用

   ```txt
   git add . //将文件夹下面的文件都添加进来
   git commit -m "提交信息" //此处提交信息可以理解为注释可以替换
   git push -u origin main //将本地仓库push到远端github上，此处要输入账号密码
   ```
